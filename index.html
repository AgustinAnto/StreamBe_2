Primer actividad de Generacion T!

Para corregir las actividades, realizar los siguentes pasos:

  • Para el punto 1 buscar '/Facil.html'
  • Para el punto 2 buscar '/Moderado-1.html'
  • Para el punto 3 buscar '/Moderado-2.html'
  • Para el punto 4 buscar '/Dificil.html'
  • Para el punto 5 buscar '/Formulario.html'
  • Para el punto 6 buscar '/Punto-6.html'
  • Para la Actividad 2 buscar '/Actividad-2.html'

Cada punto de las actividades estan separadas por paths para ser mas organizado. Un ejemplo de la correccion del punto 1 es: http://127.0.0.1:3000/index.html/Facil.html y le vendria la correccion correspondiente.
